## âš™ï¸ How It Works

1. User opens the landing page.
2. Clicks **Get Started**.
3. Fills student information form.
4. Answers interest-based questions.
5. Backend sends data to Gemini AI.
6. AI generates:
   - Career recommendations
   - Learning roadmap
   - Skills and resources
7. Results are displayed on the results page.
